### 小宇宙分享平台 | 后端开发
- 时间：2024年12月 - 2025年03月
- 项目描述：“小宇宙分享平台”是一个基于兴趣分享的社交平台，用户可以发布图文、视频等内容，分享生活点滴和兴趣爱好，并通过点赞、评论、关注等功能与其他用户互动。项目旨在打造一个温暖、有趣、高质量的社区氛围，类似小红书、Instagram 等社交平台。平台还提供会员专属权益，如会员专属内容、高级滤镜等，进一步提升用户体验。
- 技术栈：Spring Boot、Spring Cloud Alibaba、RocketMQ、Redis、Swagger、MyBatis、Docker、MySQL、Linux、OSS、Minio、SaToken
- 重点职责：
  - 基础设施模块：设计 RBAC（基于角色的访问控制），结合 SaToken 实现认证与鉴权。
  - 对象存储服务模块：使用工厂模式 + 策略模式实现文件存储扩展性，当前支持阿里云 OSS、Minio 两种类型；并通过 Nacos 分布式配置动态注册文件策略实现类 Bean 到 Spring 容器中。
  - 用户笔记模块：使用 RocketMQ 广播机制实现集群环境下笔记内容修改时本地缓存、Redis 缓存与 MySQL 数据一致性，并有效解决查询笔记详情时的缓存雪崩、穿透和击穿问题。
  - 互动模块：使用 Redis ZSET + RocketMQ 顺序消息实现用户关注/取关的高并发写与顺序性，通过 Lua 脚本保证原子性；使用 RoaringBitmap 快速判断用户是否点赞/收藏作品，并结合 RateLimiter 令牌桶实现流量削峰，保护数据库。
  - 计数与数据对齐服务模块：使用 BufferTrigger 聚合计数流量（如 1000 条/批），先写入 Redis，再通过 MQ 异步落库，并使用 XXL-Job 定时调度任务与分片广播技术实现计数数据补偿方案。
  - 会员服务模块：使用 Redis + Lua 脚本实现的分布式锁解决一人一券功能，通过定时发放优惠券吸引用户加入会员，提升用户转化率。
