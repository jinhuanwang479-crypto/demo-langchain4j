# Markdown To PDF Smoke Test

这是一份用于验证 `markdown-to-pdf` skill 的标准 Markdown 样例。

## Highlights

- 支持标题、段落和列表
- 支持 **粗体**、*斜体* 和 `inline code`
- 支持链接：[OpenAI](https://openai.com)

## Code

```java
public class Hello {
    public static void main(String[] args) {
        System.out.println("hello");
    }
}
```
